# 💼 Intern-Hub – Intelligent Recruitment & Internship Platform (Laravel)

Intern-Hub is a smart recruitment and internship connection platform built with **Laravel**. The system is designed to bridge the gap between students, interns, and employers by providing an efficient, modern, and user-friendly environment for job matching and candidate management.

The platform allows companies to post job and internship opportunities, while candidates can create professional profiles showcasing their skills, education, experience, and career goals. Intern-Hub also provides personalized job suggestions based on each candidate’s profile.

# 🚀 Key Features

## 👤 Candidate Features

### 📝 Profile Management
- ✏️ Create and update personal profile  
- 🎓 Add education background  
- 💼 Add work experience / internship history  
- 🛠️ List technical and soft skills  
- 📄 Upload CV (PDF)  
- 🎯 Set career goals and job preferences  


### 🔍 Smart Job Suggestions
- 🤖 Automatic job recommendations based on:
  - 🛠️ Skills  
  - 🎓 Major / Field of study  
  - 📈 Experience level  
  - 📍 Preferred location  
- 📊 Personalized job dashboard  


### 💼 Job Applications
- 🔎 Browse available internships and jobs  
- 🎛️ Filter by:
  - 📍 Location  
  - 💰 Salary range  
  - 🕒 Job type (Full-time / Part-time / Internship)  
- 📩 Apply directly through the platform  
- 📌 Track application status  


## 🏢 Employer Features

### 📢 Job & Internship Posting
- ➕ Create and manage job posts  
- 📝 Define:
  - 🛠️ Required skills  
  - 📈 Experience level  
  - 💰 Salary range  
  - 📄 Job description  
- ✏️ Edit / 🗑️ Delete job postings  


### 👥 Candidate Management
- 📥 View applicants for each job  
- 🔎 Filter candidates by skill and experience  
- 👤 View detailed candidate profiles  
- 🔄 Update application status:
  - ⏳ Pending  
  - ✅ Accepted  
  - ❌ Rejected

## 🛠 Admin Features

- 👥 Manage users (candidates & employers)

- 📈 Dashboard statistics:

# 🧠 Matching Logic

Intern-Hub uses a **rule-based filtering system** to recommend jobs:

- 🛠️ Match candidate skills with required job skills  
- 🎓 Compare education major with job category  
- 📈 Check experience level compatibility  
- 🏆 Rank jobs based on similarity score  

🔮 *Future versions may integrate AI-based matching algorithms for more accurate recommendations.*

# ⚠️ Limitations

- 🤖 Matching logic is rule-based (not AI-powered yet)  
- 🔔 No real-time notifications  
- 📧 No integrated email verification system (basic setup only)  
- 📊 Limited analytics for employers  
- 📱 No mobile application (web-based only)
